Sagedoku

A React application to solve sudoku puzzles

Check out the Live Demo

Check out the Github Repo

Sagedoku is a React application that uses a backtracking algorithm to solve varying difficultly levels of sudoku puzzles

This was a Hackathon project made in collaboration with 8 other developers using Git

My responsibility in this project was handling the implementation of the countdown timer and building/styling the Sudoku board

Functionalities:

  • Generate puzzle based on difficulty (easy, medium, and hard)
  • Timer to solve each puzzle
  • Option to show the solution for the puzzle
  • Option to show errors even when the puzzle is not yet completed
  • Dark and light mode
  • Mobile-friendly
